두번째 박스 모델입니다. position이란? 사용자가 원하는 위치로 이동할 수 있도록 하는 css 속성입니다. position은 위치 이동 값의 속성으로 left, right, top, bottom과 같이 사용을 하고 값으로는 static, relative, absolute, fixed가 있습니다.

세번째 박스 모델입니다. relative는 박스의 현재 있는 위치에서 사용자가 지정한 값으로 이동을 합니다. absolute는 브라우저를 기준으로 사용자가 지정한 위치 값으로 이동을 하며 float처럼 값을 지정하면 공중으로 띄어지는 현상이 생기며 scrolling했을때 위아래로 같이 scrolling이 됩니다. fixed는 absolute와 같지만 scrolling했을때 사용자가 지정한 위치가 고정되어 움직이지 않습니다. 우리는 이 박스에 class 속성을 지정하고 spacial이라는 네이밍을 작성하고 css 효과를 지정합니다. padding을 10px, 글자색을 #f00, 테두리 효과 지정, 배경색 yellow를 지정하여 이 박스에 position을 testing 해볼것 입니다.
